# Checklists

## Project Initiation Checklist

- [ ] Project approved by management
- [ ] Budget created and approved in ATLAS
- [ ] Artist agreement signed
- [ ] Producer selected and under contract
- [ ] Studio booked
- [ ] Timeline confirmed
- [ ] Team assigned
- [ ] Kickoff meeting scheduled

## Recording Session Checklist

### Before Session
- [ ] Studio confirmed and deposit paid
- [ ] Producer and engineer confirmed
- [ ] Session musicians booked (if applicable)
- [ ] AFM paperwork prepared (if applicable)
- [ ] Equipment list verified
- [ ] Artist briefing completed

### During Session
- [ ] Session notes documented
- [ ] Takes tracked
- [ ] Personnel logged
- [ ] Technical issues noted
- [ ] Budget tracking
- [ ] Daily wrap-up

### After Session
- [ ] Recording files backed up
- [ ] Session invoices received
- [ ] Payment processed
- [ ] Session documentation filed
- [ ] Next session scheduled

## Sample Clearance Checklist

- [ ] Sample source identified
- [ ] Sample percentage calculated
- [ ] Master owner located
- [ ] Publishing owner located
- [ ] Master license negotiated and executed
- [ ] Publishing license negotiated and executed
- [ ] Payment processed
- [ ] Documentation filed
- [ ] PRO notified (if applicable)
- [ ] Metadata updated

## Contract Review Checklist

- [ ] Legal review completed
- [ ] All signatures collected
- [ ] Counterpart copies exchanged
- [ ] Payment terms verified
- [ ] Rights clearly defined
- [ ] Dispute resolution included
- [ ] Insurance requirements verified
- [ ] File stored appropriately

## Budget Closure Checklist

- [ ] All invoices received
- [ ] All expenses recorded in ATLAS
- [ ] Budget vs. actual reconciliation
- [ ] Variance analysis completed
- [ ] Contingency usage documented
- [ ] Final report generated
- [ ] Lessons learned documented
- [ ] Files archived

## Metadata & Credits Checklist

- [ ] All tracks have ISRC codes
- [ ] Artist names correct (all variations)
- [ ] Genre and mood tags appropriate
- [ ] Track credits complete and accurate
- [ ] Publishing information verified
- [ ] Producer credits correct
- [ ] Engineer credits correct
- [ ] Featured artist information complete
- [ ] Metadata reviewed for QA
- [ ] System entry verified

## Release Readiness Checklist

### Audio Quality
- [ ] All masters delivered
- [ ] Loudness compliance verified
- [ ] Format versions created
- [ ] QA testing passed
- [ ] Backup copies made

### Rights & Legal
- [ ] Sample clearances obtained
- [ ] Side artist agreements executed
- [ ] Union compliance documented
- [ ] Publishing registration complete
- [ ] Copyright notices verified
- [ ] All contracts finalized

### Production & Distribution
- [ ] Metadata complete and verified
- [ ] Artwork at correct resolution
- [ ] Distribution package prepared
- [ ] Platform accounts verified
- [ ] Pre-order setup complete

### Financial
- [ ] All invoices received and paid
- [ ] Budget reconciliation complete
- [ ] Advance payments processed
- [ ] Payment obligations tracked
- [ ] Financial close-out signed off

### Approval
- [ ] A&R Administration sign-off
- [ ] Finance Department sign-off
- [ ] Legal Department sign-off
- [ ] Label Management approval
- [ ] Artist/Manager approval

## Platform Delivery Checklist

### Pre-Delivery
- [ ] Release information complete
- [ ] All metadata verified
- [ ] Artwork files prepared
- [ ] Audio files formatted correctly
- [ ] ISRC codes assigned
- [ ] UPC code obtained (physical)

### Submission
- [ ] Login credentials verified
- [ ] Release information uploaded
- [ ] Audio files uploaded
- [ ] Metadata mapped correctly
- [ ] Artwork uploaded (all sizes)
- [ ] Release date scheduled

### Post-Delivery
- [ ] Verification on platform
- [ ] Link generation
- [ ] Social media sharing prepared
- [ ] Analytics tracking setup
- [ ] Performance monitoring begins

- [ ] Vinyl master created
- [ ] Test pressing ordered
- [ ] Test pressing approved
- [ ] Artwork finalized
- [ ] Pressing plant selected
- [ ] Order placed
- [ ] Production timeline confirmed
- [ ] Delivery date scheduled
- [ ] Inventory receipt verified
- [ ] Distribution arranged
