# Synchronization

## Overview

Synchronization (sync) licensing grants permission to use compositions in visual media and is a major revenue source for publishers.

## Sync License Basics

### What It Covers
- Film and TV
- Video games
- Advertising/commercials
- Documentaries
- YouTube videos
- Podcasts
- Website background music

### What It Doesn't Cover
- Master recording (separate license)
- Performance rights (separate PRO)
- Mechanical reproduction (separate)

## Types of Sync Licenses

### Master Use License
- Permission from master owner (label)
- For specific recording
- Can't use different version
- Negotiated separately

### Composition Sync License
- Permission from publisher
- Any version of song
- Covers composition only
- Granted by publisher

## Sync Licensing Process

### Step 1: Identification
- Composition identification
- Publisher discovery
- License requirement verification

### Step 2: Inquiry
- Submission to publisher
- Usage type description
- Territory and term
- Quote request

### Step 3: Negotiation
- Rate negotiation
- Term agreement
- Territory limitations
- Approval conditions

### Step 4: Execution
- License agreement signing
- Payment processing
- Term documentation
- File storage

## Sync Royalty Structures

### Upfront Fee
- One-time payment
- Varies greatly: $100-$50,000+
- Depends on use and media
- Non-refundable typically

### Backend Royalties
- Percentage of receipts
- TV/film: Often not included
- Streaming: Increasingly common
- Tracking and reporting required

### Factors Affecting Rate
- Media type (film, TV, ad)
- Territory (local vs. international)
- Duration of use (feature film vs. 15-sec ad)
- Exclusivity (exclusive vs. non-exclusive)
- Artist prominence

## Sync Clearance Checklist

- [ ] Both licenses obtained (master + composition)
- [ ] Territory confirmed
- [ ] Term length appropriate
- [ ] Exclusivity noted
- [ ] Approval conditions met
- [ ] Payment processed
- [ ] Agreement stored
- [ ] Credits/usage documented
