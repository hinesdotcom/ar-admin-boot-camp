# Label Copy

## Overview

Label copy is the textual information that appears on physical album packaging and digital storefronts, including credits and copyright information.

## Label Copy Elements

### Front Cover
- Album title
- Artist name
- Artwork
- Label logo
- Format indication

### Back Cover
- Track listing
- Personnel and credits
- Production notes
- Artwork credits
- Copyright information

### Spine
- Album title
- Artist name
- Label name
- Barcode

### Digital Storefronts
- Album description
- Genre and mood tags
- Featured artist listings
- Release notes
- Copyright line

## Credits Section

### Standard Format
- Song title
- Songwriter(s)
- Producer(s)
- Feature artist(s)
- Engineer(s)
- Copyright line

### Example Credit Line
```
"Song Title"
Written by: John Smith, Jane Doe
Produced by: Producer Name
Featuring: Guest Artist
Engineered by: Engineer Name
© 2024 Label Name
```

## Legal Requirements

### Copyright Notice
- Copyright symbol (©)
- Year of publication
- Copyright owner name
- All rights reserved statement

### Mechanical Notice
- Mechanical licensing information
- Harry Fox Agency reference
- Mechanical payment notation

### Performance Rights
- PRO affiliation (ASCAP, BMI, SESAC)
- Songwriter identifications
- Performer information

## Production Notes

- Recording location
- Recording dates
- Personnel thanks
- Producer notes
- Special credits
- Acknowledgments

## Approval Process

1. Draft creation
2. Legal review
3. Artist approval
4. Final layout
5. Printing/digital delivery
6. Archive
