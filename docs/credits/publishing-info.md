# Publishing Info

## Overview

Publishing information identifies songwriters, publishers, and ownership percentages for proper royalty distribution and rights management.

## Publishing Data Elements

### Songwriter Information
- Full legal name
- PRO affiliation (ASCAP, BMI, SESAC)
- PRO confirmation number
- Writer share percentage
- Role (composer, lyricist, both)

### Publisher Information
- Publisher name
- Publisher PRO affiliation
- Publisher confirmation code
- Publisher share percentage
- IPI number

### Splits & Ownership
- Writer split percentage
- Publisher split percentage
- Administration territory
- Reversion conditions
- Collection society routing

### Registrations

#### PRO Registration (ASCAP/BMI/SESAC)
- Song title
- Songwriter names and shares
- Publisher names and shares
- Registration confirmation
- Effective date

#### Mechanical Licensing
- Harry Fox Agency registration
- MLC (Mechanical Licensing Collective) registration
- Mechanical reporting requirements
- Payment collection

## Publishing Split Example

```
Song Title: "Example Song"

Writers:
- Writer A (John Smith): 50%
  PRO: ASCAP (123456789)
- Writer B (Jane Doe): 50%
  PRO: BMI (987654321)

Publishers:
- Publisher A: 50%
- Publisher B: 50%
```

## Registration Process

1. Gather all information
2. Verify splits accuracy
3. Songwriter approval
4. PRO submission
5. Confirmation number tracking
6. MLC registration (if applicable)
7. File archival

## Common Issues

- Incomplete split documentation
- Wrong PRO affiliations
- Missing confirmation numbers
- Incorrect split percentages
- Late registrations
- Administrative disputes

## Reporting & Payments

- Quarterly royalty statements
- Mechanical payment tracking
- Performance royalty distribution
- Sync licensing royalties
- Annual reconciliation
