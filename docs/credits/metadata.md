# Metadata

## Overview

Metadata is the informational data associated with music tracks that enables proper identification, crediting, and payment distribution.

## Essential Metadata Fields

### Identification
- Track title (with version info)
- Artist name (main)
- Album/project title
- Release date
- ISRC code (unique identifier)

### Technical
- Duration
- Recording date
- Format and bitrate
- File naming convention
- Version number

### Personnel
- Producer name
- Songwriter(s)
- Engineer(s)
- Featured artists
- Additional credits

### Publishing
- Publishing ownership percentages
- Publishing account codes
- Songwriter splits
- Co-publishing information
- Performance rights organization (PRO)

### Rights
- Copyright owner
- Master owner
- Distribution rights
- Territory limitations
- Usage restrictions

## Metadata Standards

### DDEX Standards
- Digital release information exchange
- Standardized format
- International compatibility
- Platform delivery

### ID3 Tags
- MP3 metadata format
- Album art embedding
- ISRC and ISWC codes
- Composer and performer info

### RIAA Standards
- Recording identification
- Format specifications
- Delivery requirements

## Metadata Management

### Creation Process
1. Gather all information
2. Verify accuracy
3. Format standardization
4. System entry
5. QA review
6. Delivery to platforms

### Quality Assurance
- Spelling accuracy
- Consistency verification
- Encoding validation
- Format compliance
- Completeness check

## Common Metadata Errors

- Misspelled artist names
- Incorrect split percentages
- Missing ISRC codes
- Wrong album associations
- Incomplete publishing info
- Credit omissions
