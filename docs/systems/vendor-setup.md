# Vendor Setup

## Overview

Establishing and managing vendor relationships is critical for A&R administration success.

## Types of Vendors

### Recording Services
- Recording studios
- Engineers
- Producers
- Session musicians

### Post-Production
- Mixing engineers
- Mastering studios
- Quality assurance services

### Clearance Services
- Sample clearance specialists
- Rights investigation services
- Mechanical licensing

### Distribution & Delivery
- Digital aggregators
- Physical distributors
- Metadata providers

## Vendor Onboarding Process

### Step 1: Vetting
- Capabilities assessment
- References check
- Pricing negotiation
- Contract terms

### Step 2: Setup
- Account creation
- Payment method registration
- Service level agreements
- Contact assignment

### Step 3: Integration
- System setup
- Process documentation
- Communication protocol
- Issue escalation path
