# UniPort

## Overview

UniPort is the central music data and metadata management system, serving as the single source of truth for all release information.

## Key Features

- Release information database
- Metadata management
- Rights tracking
- Distribution coordination
- Reporting and analytics

## Accessing UniPort

### Login Credentials
- SSO enabled
- Role-based access controls
- Department-specific permissions

### User Roles
- Admin - Full access
- Manager - Project management
- Contributor - Data entry
- Viewer - Read-only

## Common Workflows

### Creating a Release Record
1. New release entry
2. Basic information
3. Track details
4. Artist information
5. Rights assignment

### Managing Metadata
1. ISRC codes
2. Publishing information
3. Split information
4. Alternate titles

### Tracking Rights
1. Sample clearance status
2. Rights holder contacts
3. Licensing agreements
4. Expiration tracking

## Best Practices

- Update metadata regularly
- Use standardized naming conventions
- Maintain audit trail
- Regular backups
