# ATLAS

## Overview

ATLAS is the core financial management system used for budget tracking, vendor payments, and financial reporting.

## Key Features

- Budget creation and tracking
- Vendor management
- Payment processing
- Financial reporting
- Project cost analysis

## Accessing ATLAS

### Login
- Enterprise single sign-on
- Multi-factor authentication required
- Account setup through IT Department

### Navigation
- Dashboard overview
- Project search
- Budget modules
- Reporting suite

4. Approval workflow

### Processing Payments
1. Invoice receipt
