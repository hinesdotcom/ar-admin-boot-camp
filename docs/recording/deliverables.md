# Deliverables

## Overview

Deliverables are the final outputs from the recording, mixing, and mastering processes that are ready for distribution.

## Master Audio Deliverables

### Stereo Master
- Final mixed and mastered stereo file
- WAV format, 24-bit/96kHz minimum
- Loudness metering data
- Metadata embedded

### Mono Mix
- Downmix for compatibility
- WAV format, 24-bit/96kHz
- Loudness verification
- Mono compatibility check

### Instrumental Version
- All vocal elements removed
- Master quality standards
- Separate file per track
- Full metadata

### Acappella Version
- Vocals only, no instrumental
- Master quality standards
- For remix/karaoke use
- Separate file per track

## Format Specifications

### WAV Format
- 24-bit minimum depth
- 96kHz sample rate
- Linear PCM
- No metadata embedding

### MP3 Format
- 320 kbps
- Metadata tagged
- Reference listening
- Quality assurance verification

### Hi-Res Formats
- FLAC 24/96 or 24/192
- DSD format options
- Lossless compression
- Archival quality

## Technical Specifications

### Loudness Metering
- LUFS measurement
- True peak metering
- Integrated loudness
- Format-specific standards

### Metadata Requirements
- Track title
- Artist name
- Album information
- ISRC code
- Copyright information
