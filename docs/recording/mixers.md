# Mixers

## Overview

Mixers are technical and creative experts who balance and enhance recorded elements to create the final stereo master.

## Mixing Process

### Preparation
- Receive raw tracks
- Review producer notes
- Establish reference samples
- Session setup

### Initial Balance
- Level setting
- Panning basics
- Basic processing
- Rough mix creation

### Mixing Phase
- Detailed processing
- EQ and compression
- Effects and automation
- Vocal treatment
- Instrument balancing

### Reference & Revision
- Comparison to references
- Revision implementation
- Multiple pass-throughs
- Client feedback

### Final Delivery
- Mix stems exported
- Instrumental version
- A cappella version
- Final master mix

## Mixer Selection

### Criteria
- Genre expertise
- Previous work quality
- Studio capabilities
- Timeline availability
- Budget fit
- Communication style

### Negotiation
- Per-track fee
- Revision limit
- Delivery timeline
- Format requirements
- Backup sessions

## Mixer Agreement Components
