# Mastering

## Overview

Mastering is the final technical step that prepares mixes for distribution across all platforms and formats.

## Mastering Process

### Preparation
- Receive final stereo mix
- Reference track selection
- Target platform analysis
- Loudness requirements research

### Analysis Phase
- Mix evaluation
- Frequency balance assessment
- Dynamic range analysis
- Format compatibility check

### Mastering Session
- EQ and compression
- Limiting and loudness optimization
- Stereo imaging
- Format-specific versions
- Reference monitoring

### Quality Control
- AB comparison with references
- Multi-format testing
- Loudness metering
- Clarity and translation check
- Final adjustments

### Delivery
- Master file export
- Format-specific versions
- Loudness compliance
- Metadata integration
- Archive backup

## Mastering Engineer Selection

### Evaluation

## Loudness Standards

### Streaming Platforms
- Spotify: -14 LUFS
- Apple Music: -16 LUFS
- YouTube: -13 LUFS
- General: -14 LUFS target

### Physical Media
- CD: -6 to -8 dBFS
- Vinyl: Format-specific
- Hi-Res: -3 dBFS
