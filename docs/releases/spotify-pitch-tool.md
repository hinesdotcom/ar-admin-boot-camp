# Spotify Pitch Tool

## Overview

The Spotify for Artists Pitch Tool enables direct pitching to Spotify curators for playlist consideration.

## Access Requirements

### Setup
- Spotify for Artists account
- Verified artist profile
- Admin access to artist profile
- At least 50 followers (eligibility varies)

### Account Verification
- Email verification
- Profile linking
- Payment information (if required)
- Profile completeness

## Pitch Submission Process

### Step 1: Access Pitch Tool
- Log into Spotify for Artists
- Navigate to Pitching section
- Start new pitch

### Step 2: Select Release
- Choose track or album
- Release date specification
- Format selection
- Category assignment

### Step 3: Pitch Details
- Genre selection
- Mood and energy tags
- Track narrative (500 characters)
- Links to reference tracks
- Artist bio

### Step 4: Submission
- Campaign name
- Target playlists (if applicable)
- Submission date
- Final review and confirmation

## Pitch Best Practices

### Narrative Writing
- Compelling, concise description
- Story and context
- Artist background
- Inspiration and influences
- Why this playlist fits

### Content Optimization
- Accurate genre selection
- Appropriate mood tagging
- Quality artwork
- Complete metadata
- Professional presentation

### Reference Tracks
- Genre-matching selections
- Mood and energy alignment
- Artist level comparison
- Playlist context

## Timeline Expectations

- Pitch submission: Up to 4 weeks before release
- Curator review: 2-4 weeks
- Playlist decision: 1-2 weeks before release
- Playlist placement: Release date or shortly after

## Pitch Results

### Success Metrics
- Playlist placements
- Listener additions
- Stream growth
- Algorithmic recommendations

### Analytics Tracking
- Pitch performance dashboard
- Listener insights
- Geographic distribution
- Streaming trends

## Rejection Handling

- Request feedback (if available)
- Analyze playlist fit
- Adjust narrative and metadata
- Resubmit for future releases
- Build follower base
