# Production Delivery

## Overview

Production delivery is the transfer of final master files and metadata to distribution partners and streaming platforms.

## Delivery Requirements

### File Specifications
- Format: WAV, FLAC, or MP3
- Bit depth: 24-bit minimum
- Sample rate: 96 kHz minimum
- Channel configuration: Stereo (or specified format)
- File naming: Standard convention

### Metadata Delivery
- Complete metadata spreadsheet
- ISRC codes for all tracks
- Album artwork (minimum 3000x3000px)
- Genre and mood tags
- Featured artist information

### Supporting Documentation
- Master recording agreement
- Rights verification
- Artist approval
- Quality assurance report
- Technical specifications sheet

## Distribution Partners

### Primary Aggregators
- DistroKid
- TuneCore
- CD Baby
- Stem
- UnitedMasters

### Direct Delivery
- Spotify Direct
- Apple Music Direct
- YouTube Music

## Delivery Process

### Step 1: Preparation
- File format conversion
- Metadata compilation
- Artwork preparation
- Documentation review

### Step 2: Upload
- Platform login
- Release information entry
- File upload
- Metadata mapping
- Artwork assignment

### Step 3: Verification
- File validation
- Metadata review
- Platform preview
- QA sign-off

### Step 4: Scheduling
- Release date confirmation
- Pre-order activation
- Launch monitoring
- Performance tracking

## Platform-Specific Requirements

### Spotify
- ISRC requirement
- Artist URL verification
- Explicit content flagging
- Playlist pitching integration

### Apple Music
- ISRC requirement
- Artwork compliance
- Metadata accuracy
- Media Mail setup

### YouTube Music
- Content ID setup
- Artist channel linking
- Metadata optimization
- Playlist consideration
