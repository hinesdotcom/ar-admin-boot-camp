# Budgets

## Overview

Budget management is fundamental to project profitability and financial control. All projects must have approved budgets before production begins.

## Budget Components

### Recording Costs
- Studio time
- Engineering fees
- Producer payments
- Session musician fees

### Post-Production
- Mixing engineer fees
- Mastering costs
- Quality assurance
- Format delivery

### Clearances & Rights
- Sample clearance costs
- Mechanical licenses
- Talent releases
- Union payments

### Administrative
- Project management overhead
- Vendor coordination
- Documentation
- Contingency reserves

## Budget Process

### Step 1: Estimation
- Gather quotes from vendors
- Research comparable projects
- Factor in contingencies
- Add administrative overhead

### Step 2: Review & Approval
- Department head review
- Finance sign-off
- Budget owner confirmation
- Project approval

### Step 3: Tracking
- Record all expenses
- Compare to budget
- Monitor variances
- Generate reports

### Step 4: Closure
- Final reconciliation
- Variance analysis
- Lessons learned
- Archive documentation

## Budget Best Practices

- Always get multiple quotes
- Include 10-15% contingency
- Track actual spending weekly
- Flag variances immediately
- Document all changes
