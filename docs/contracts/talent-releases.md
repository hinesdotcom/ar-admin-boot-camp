# Talent Releases

## Overview

Talent releases are agreements signed by performers, granting the label rights to use their performances in recordings and related media.

## Release Components

### Identification
- Talent name and contact
- Performance description
- Recording date
- Project identification

### Grant of Rights
- Master recording rights
- Performance rights assignment
- Derivative work rights
- Perpetual rights duration

### Compensation
- Session payment amount
- Union scale compliance
- Bonus/royalty terms
- Expenses reimbursement

### Use Restrictions
- Permitted uses
- Prohibited uses
- Media limitations
- Territory scope

### Credits & Publishing
- Album credit line
- Metadata requirements
- Publishing percentages
- PRO registration

### Representations
- Talent warranty of rights
- No conflicts representation
- Authority to sign
- Fitness of performance

## Talent Release Types

### Session Musician Release
- One-time recording session
- Limited use rights
- Session fee basis
- Union compliance

### Featured Artist Release
- Lead vocal or prominent performance
- Broader use rights
- Publishing participation
- Royalty consideration

### Backup Artist Release
- Supporting performance
- Standard fee-based
- Limited credit
- Simplified terms

## Key Clauses

- Confidentiality
- Indemnification
- Insurance requirements
- Union obligations
- Termination conditions

## Processing

- Talent identification and contact
- Rate negotiation
- Release preparation
- Signature collection
- Payment processing
- File archival
